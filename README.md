# A/B Testing Analysis Project

### Project Overview
This project focuses on analyzing A/B test results to identify statistically significant uplifts in key business metrics across various segments (countries, channels, and devices).

### Key Features
* **Data Processing:** Python (Pandas) for data cleaning and transformation.
* **Statistical Analysis:** Calculation of Uplift and Statistical Significance (p-value).
* **Visualization:** Interactive Tableau Dashboard for granular insight discovery.
